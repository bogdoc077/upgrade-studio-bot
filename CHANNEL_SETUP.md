# Налаштування бота для роботи з каналами

## 1. Налаштування Privacy Mode через BotFather

1. Відкрийте @BotFather в Telegram
2. Виберіть `/mybots`
3. Виберіть вашого бота `@UpgradeStudioFitnessBot`
4. Натисніть `Bot Settings`
5. Натисніть `Group Privacy`
6. Виберіть `Turn off` (це дозволить боту бачити всі повідомлення в групах/каналах)

## 2. Додавання бота до каналу

1. Перейдіть до вашого каналу
2. Натисніть на назву каналу (вгорі)
3. Виберіть `Manage Channel` або `Налаштування каналу`
4. Виберіть `Administrators` або `Адміністратори`
5. Натисніть `Add Admin` або `Додати адміна`
6. Знайдіть `@UpgradeStudioFitnessBot`
7. Дайте боту права:
   - ✅ Post Messages (Надсилати повідомлення)
   - ✅ Edit Messages (Редагувати повідомлення)
   - ✅ Delete Messages (Видаляти повідомлення)

## 3. Тестування

Після налаштування спробуйте:
1. Написати будь-що в каналі - має з'явитися логування
2. Використати команду `/chat_id` в каналі
3. Перевірити логи в терміналі бота

## 4. Команди для каналів

- `/chat_id` - отримати ID каналу (результат у логах + приватне повідомлення)
- `/get_chat_info` - детальна інформація (тільки для груп і приватних чатів)

## Діагностика

Якщо бот не реагує:
1. Перевірте, чи вимкнений Privacy Mode
2. Переконайтеся, що бот є адміністратором каналу
3. Перевірте права бота в каналі
4. Подивіться логи в терміналі - має бути запис про кожне повідомлення
