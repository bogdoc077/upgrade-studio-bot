# Додавання відео-привітання (кружечок)

## Як додати відео-привітання

1. **Запишіть відео-кружечок:**
   - Відкрийте Telegram на телефоні
   - Натисніть і утримуйте кнопку запису відео в чаті
   - Запишіть коротке (30-60 секунд) відео-привітання
   - Збережіть відео

2. **Отримайте файл відео:**
   - Надішліть відео-кружечок собі в "Збережені повідомлення"
   - Завантажте файл на комп'ютер
   - Перейменуйте на `welcome_video.mp4`

3. **Додайте до проекту:**
   - Покладіть файл в папку `assets/welcome_video.mp4`
   - Розкоментуйте код в `main.py` в функції `send_welcome_intro()`

## Код для відео-привітання

В файлі `main.py` в функції `send_welcome_intro()` розкоментуйте:

```python
# TODO: Додати відео-привітання (кружечок)
video_path = "assets/welcome_video.mp4"
if os.path.exists(video_path):
    await update.message.reply_video_note(
        video_note=open(video_path, "rb"),
        reply_markup=get_welcome_keyboard()
    )
```

## Додаткові налаштування

Також не забудьте додати import:
```python
import os
```

## Альтернативний варіант

Якщо відео-файл великий, можете:
1. Завантажити відео на YouTube/Vimeo
2. Надсилати посилання разом з текстом
3. Або використовувати GIF-анімацію замість відео

## Формати підтримуваних файлів

- `.mp4` (рекомендований)
- `.mov` 
- `.avi`
- Максимальний розмір: 50MB для Telegram Bot API
